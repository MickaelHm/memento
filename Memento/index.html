Memento HTML & CSS

HTML

Il faut vérifier que la balise <meta charset="UTF-8" /> est bien présente entre les balises head afin de pouvoir afficher correctement les accents présent dans la langue française.
Entre les balises head ne se trouve que des éléments non affiché sur la page web.
Entre les balises body se trouve les éléments affiché sur la page web.
Pour afficher du code sur une page HTML on remplace le premier chevron par "& lt ;" sans espace et le second par "& gt ;"
Les paragraphes se trouvent dans des balises <p> </p>
Dans la balise <Title> on écrit le titre qui figurera au niveau de l'onglet du navigateur
Chaque Titre affiché sur la page se trouvera entre des balises h1/h2/h3... suivant son importance, h1 étant le plus important. Cela influe sur sa taille.
Afin de revenir à la ligne on terminera sa ligne par une balise "br /".
Pour créer des listes, on peut utiliser les balises "ul" pour une liste non numérotée et "ol" pour une numérotée. Elle devront être suivit par des balises "li" "/li" pour élément de la liste/
Des balises pour la mise en page tel que "b" pour écrire en gras ou "i" pour écrire en italic existe, cette liste est non exhaustive.
L'on peut attribuer des couleurs entre autre via une balise "style" directement depuis la page HTML (ou comme on le verra plus tard, s'en passer en utilisant le CSS pour ces applications). Il est possible d'affecter qu'un morceau d'une phrase grace a une balise de délimitation "span" "/span" comme pour colorer un seul mot dans une phrase par exemple. ex: <span style="color:green;"> litres</span>
Pour insérer une image à partir d'une URL on utilisera la balise <img src ="URL"/>
Dans l'exemple qui suit, on utilise la balise "a href" pour mettre un lien qui s'affiche sur le mot que l'on désire, dans cette le phrase, on cliquera sur "ici" pour ouvrir le lien en question;
<a href="https://www.codecademy.com/fr" >ici</a>
En prenant le même exemple, on peut rajouter une balise title qui permettra d'afficher un message au survol du lien:
Le site de codeacademy est <a href="https://www.codecademy.com/fr" title="mettre un titre au survol du lien"/>ici</a/>
Pour ajouter des commentaires en HTML la balise à utiliser est celle ci: <!-- Mon Commentaire -->

Tableau

Pour la création d'un tableau on utilisera table{} intégré dans une balise style, auquel on peut spécifier le type de bordure, comme par exemple "border-collapse: collapse;" qui sert a fondre les lignes du tableau entre elles.
Pour ajouter un en-tête à notre tableau on utilise la balise "caption".
Un tableau est structuré sous forme de <thead> , <tbody> , <tfoot> .
On assigne une taille de bordure et une couleur de cette manière "border: 1px solid black;" aux rubriques voulu du tableau, définient comme tel :

  • "table" pour le contour du tableau
  • "th" pour la première ligne du tableau
  • "td" pour toutes les cases sous la première ligne
Dans cet exemple colspan="2" inséré dans <td> indique que l'on veut que cette case fasse la taille des "2" premières colonnes:
<td colspan="2"> <center>Solde </center></td>
Les balises center servent naturellement à centrer le texte dans la case.

Formulaire

Pour créer un formulaire nous allons écrire notre code dans la balise <form>, elle sert a collecter des données entrées par l'utilisateur.
L'élément <fieldset> permet de regrouper une partie d'un formulaire HTML et d'associer une légende (<legend>) décrivant ce groupe. Exemple:


    <fieldset class="id">

        <legend class="id">Votre identité</legend>

        On peut constater la présence de l'attribut class= qui permet d'attribuer un nom à un ensemble de code afin de l'identifier et pouvoir modifier ce block pars le bias du CSS.
